# LaunchKit 🚀

> The free, open-source alternative to ShipFast.  
> Built for Indian developers with Razorpay + AI included.

![License](https://img.shields.io/badge/license-MIT-green)
![Stars](https://img.shields.io/github/stars/afaizalam2003/launchpad-pro)

## ✨ Features

- 🔐 **Auth** — Email, Google OAuth, Magic Link via Supabase
- 💳 **Payments** — Stripe + Razorpay (India) subscriptions
- 🤖 **AI Ready** — OpenAI SDK pre-wired with streaming
- 🗄️ **Database** — Supabase PostgreSQL with RLS
- 📧 **Email** — Transactional emails via Resend
- 🚀 **CI/CD** — GitHub Actions + Vercel deployment

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React + Vite + TypeScript |
| Styling | Tailwind CSS + shadcn/ui |
| Database | Supabase (PostgreSQL) |
| Auth | Supabase Auth |
| Payments | Stripe + Razorpay |
| AI | OpenAI API |
| Deploy | Vercel |

## 🚀 Quick Start
```bash
git clone https://github.com/afaizalam2003/launchpad-pro
cd launchpad-pro
npm install
cp .env.example .env.local
npm run dev
```

## ⚙️ Environment Variables
```env
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
VITE_STRIPE_PUBLISHABLE_KEY=
VITE_RAZORPAY_KEY_ID=
VITE_OPENAI_API_KEY=
```
